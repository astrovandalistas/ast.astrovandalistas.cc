<strong>Hardware hacking urbano</strong>

1. Se trabaja sobre la estructura metálica que sostiene las lonas  promocionales afuera del teatro Julio Castillo. La estructura metálica  se interviene agregando ocho filas de varillas de hierro de distintos  calibres; cada una de estas varillas genera una resonancia específica,  característica que determinará el tipo de ensamble sonoro que se produce  a partir de la percusión de estas barrillas.

2. Las varillas tienen un sistema mecánico de temporalización, esto  quiere decir que el movimiento de las mismas no se genera de forma  unísona, sino que cada uno tiene un pequeño retraso en relación con las  otras al momento de percutir sobre la estructura metálica base.

    ![](/assets/posts/bang_dibujo.jpg)

3. Utilizando un microcontrolador con ethernet se generará un  puente de comunicación entre nuestro hardware y un software de busqueda en twitter  que detonará el proceso sonoro con el motor.
